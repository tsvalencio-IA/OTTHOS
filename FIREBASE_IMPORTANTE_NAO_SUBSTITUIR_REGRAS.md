# IMPORTANTE — PROJETO FIREBASE COMPARTILHADO

O projeto `lerunnersmap` pode já possuir regras de outros aplicativos.

**Não substitua todas as regras existentes** pelo arquivo do OTTHOS. Isso pode bloquear os outros sistemas.

Abra as regras atuais e adicione o conteúdo de `firebase-rules-OTTHOS-TRECHO-PARA-MESCLAR.json` dentro do objeto principal `rules`, mantendo todos os caminhos que já existem.

O arquivo `firebase-database.rules.json` é apenas um exemplo completo para um banco exclusivo do OTTHOS.
